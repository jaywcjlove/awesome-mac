✦ 나의 macOS 애플리케이션:

읽기 및 쓰기 도구

텍스트 편집기

  • Aurora Editor - macOS용 경량 코드 편집기 (IDE).
  • Bootstrap Studio - Bootstrap 프레임워크를 사용하여 반응형 웹사이트를 제작할 수 있는 강력한 데스크톱 앱.
  • Nova - Panic에서 만든 아름답고 빠르고 유연한 네이티브 Mac 코드 편집기.
  • Plain Text Editor - 단순하고 집중을 방해하지 않는 메모장.
  • Sublime Text - 플러그인 관리 시스템을 갖춘 인기 있고 깔끔한 편집기.
  • SubEthaEdit - 쓰기, 코딩 및 협업을 위한 강력한 편집기.
  • TextMate - 텍스트 편집기 세계에 Apple의 운영 체제 접근 방식을 도입한 편집기.
  • Tot - 텍스트를 수집하고 편집하는 우아하고 간단한 방법.
  • Vim - 오래된 터미널 기반 편집기.
  • Vimr - OS X를 위해 정제된 Vim 경험.
  • Zed - Atom과 Tree-sitter 제작자들이 만든 고성능 멀티플레이어 코드 편집기.

오피스

  • Keynote - 멋진 프레젠테이션 제작.
  • LibreOffice - 대규모 커뮤니티에서 매일 사용하고 테스트하는 무료 오픈 소스 오피스 소프트웨어.
  • Microsoft Office - Mac용으로 설계된 확실한 Office.
  • Numbers - 인상적인 스프레드시트 제작.
  • OnlyOffice - 텍스트, 스프레드시트 및 프레젠테이션 편집기를 결합한 오피스 스위트.
  • Pages - 돋보이는 문서 제작.
  • SoftMaker Office - Microsoft Office 문서와 완전한 호환성을 목표로 하는 오피스 스위트.
  • WPS - 교차 플랫폼 오피스 소프트웨어 스위트.

마크다운 도구

  • Archimedes - 인라인 LaTeX를 지원하는 수학 쓰기에 최적화된 마크다운 편집기.
  • EME - Chrome과 같은 인터페이스를 가진 오픈 소스 마크다운 편집기.
  • iA Writer - 단순함과 디자인에 강조를 둔 쓰기 앱.
  • LightPaper - 단순하고 아름다우면서도 강력한 Mac용 텍스트 편집기.
  • Marked 2 - 작가들을 위한 우아하고 강력한 도구 세트를 갖춘 마크다운 미리보기.
  • MarkText - 차세대 마크다운 편집기.
  • Marp - 교차 플랫폼을 지원하는 마크다운 프레젠테이션 작성기.
  • Marxico - Evernote를 위한 섬세한 마크다운 편집기.
  • MWeb - 프로페셔널 마크다운 쓰기 및 정적 블로그 생성 앱.
  • Obsidian - 당신을 위한 영원한 제2의 뇌.
  • Typora - 원활한 라이브 미리보기를 제공하는 미니멀 마크다운 편집기.
  • Ulysses - Mac, iPad, iPhone을 위한 궁극의 쓰기 앱.
  • Zettlr - 21세기를 위한 마크다운 편집기.

  • Noteship - 노트를 지식으로 바꿉니다. 오프라인으로 작동하며 로컬에 저장됩니다.
