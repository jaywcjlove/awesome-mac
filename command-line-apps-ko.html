훌륭한 애플리케이션 (Awesome Command Line Apps)

엄선된 유용한 명령줄 애플리케이션 목록

✦ 나의 macOS 애플리케이션:

데이터베이스

  • mycli - 자동 완성 및 구문 강조 기능이 있는 MySQL용 CLI.
  • pgcli - 자동 완성 및 구문 강조 기능이 있는 Postgres용 명령줄 인터페이스.

미디어

  • cmus - 명령줄 기반의 음악 플레이어.
  • youtube-dl - YouTube.com 및 기타 사이트에서 비디오를 다운로드하는 도구.

개발자 도구

  • httpie - 당신을 미소 짓게 할 명령줄 HTTP 클라이언트.
  • JSON Schema CLI - JSON 스키마 작업을 위한 CLI. 포맷팅, 린팅, 테스트, 번들링 등을 지원하며 로컬 개발 및 CI/CD 파이프라인에서 사용 가능.
  • mdctl - 마크다운 파일 처리를 위한 명령줄 도구. 이미지 다운로드, 파일 번역 등 지원.
  • SimTool - 강력한 앱 브라우징 및 파일 보기 기능을 갖춘 iOS 시뮬레이터 관리용 터미널 UI.

기타

  • autojump - 길고 복잡한 경로 대신 한 번의 점프로 이동하는 도구.
  • bash-it - 커뮤니티 기반의 bash 프레임워크.
  • bat - 구문 강조 및 Git 통합 기능이 있는 cat(1) 클론.
  • bclm - 배터리 최대 충전량을 제한하는 macOS 명령줄 도구.
  • cool-retro-term - 향수를 불러일으키는 빈티지 스타일의 터미널.
  • Cakebrew - Homebrew를 위한 GUI 클라이언트. 명령어를 사용하지 않고도 소프트웨어 설치, 확인, 제거 가능.
  • Fish Shell - 지능적이고 사용자 친화적인 명령줄 쉘.
  • Glances - 명령줄에서 시스템 리소스 상태를 확인하는 도구.
  • m-cli - macOS를 위한 스위스 아미 나이프 같은 도구.
  • mas - 간단한 Mac App Store 명령줄 인터페이스.
  • thefuck - 이전 명령의 오타를 수정해주는 유용한 도구.
  • tmux - 강력한 터미널 멀티플렉서.
  • trash - 파일과 디렉토리를 휴지통으로 이동시키는 도구.
  • Zsh - 대화형 사용을 위해 설계된 명령줄 쉘.
  • ohmyzsh - zsh 설정을 관리하는 커뮤니티 주도 오픈 소스 프레임워크.
