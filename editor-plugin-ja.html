Sublime Text プラグイン

  • PlainTasks - こだわりのTodoリストプラグイン。
  • sublimelint - エラーハイライト用フレームワーク。AppleScript、C、C++、CSS、CoffeeScript、Go、HAML、HTML、Java、JavaScript、Lua、NASM、PHP、Perl、Python、Puppet、Ruby、XMLに対応。
  • jQuery - Sublime Text用jQueryパッケージバンドル。
  • Underscore - Sublime Text 2用Underscore 1.4スニペット。
  • DocBlockr - Javascript、PHP、CoffeeScript、ActionScript、C、C++でDocBlockコメントの記述を簡素化。
  • SublimeCodeIntel - フル機能のコードインテリジェンスとスマートオートコンプリートエンジン。
  • ColorPicker - マルチプラットフォーム対応のカラーピッカープラグイン。
  • HTML-CSS-JS Prettify - node.jsによるSublime Text 2/3用HTML、CSS、JavaScript、JSONコードフォーマッター。
  • WakaTime - プログラミング活動から自動的にメトリクス、インサイト、時間追跡を生成するプラグイン。
  • Seti_UI - ファイルアイコンを含む美しいテーマ。
  • OmniMarkupPreviewer - マークアップファイル（markdown、rst、creole、textile...）のライブプレビュー/エクスポーター。
  • BracketHighlighter - 括弧とタグのハイライター。
  • Markdown Extended - Markdownシンタックスハイライター。
  • Babel - React JSX拡張付きES6 JavaScriptのシンタックス定義。
  • AutoFileName - ファイル名を自動補完。
  • FTPSync - FTP同期プラグイン、仮想マシンとのローカルファイル同期に非常に便利。
  • Color Highlighter - カラーキーワードをハイライト、CSSコードに最適。
  • File History - ローカルファイル履歴、ファイルを変更するたびに履歴が保存される。
  • JavaScript Completions - ネイティブJavaScript補完。
  • JSON Reindent - JSONフォーマッティング。
  • Git - Sublime Text用の包括的なGitプラグイン。
  • GitStatus - ファイルの変更とプロジェクトの状態をリアルタイムで監視。
  • Gitignore - Git Gitignoreファイルを生成、よく使われるビルトインフィルターファイル。

VSCode プラグイン

  • 共通

    • Auto-Open Markdown Preview - Markdownファイルを開くとMarkdownプレビューを自動的に開く
    • Bracket Pair Colorizer - VSCode用括弧カラーライザー拡張機能
    • Code Spell Checker - TypeScriptとJavaScript用のシンプルなソースコードスペルチェッカー
    • Debugger for Chrome - Google ChromeブラウザでJavaScriptコードをデバッグ。
    • Document This - Visual Studio Code用JSDocジェネレーター拡張機能
    • ESlint - ESLintをVSCodeに統合。
    • Git Lens - Visual Studio Code内蔵のGit機能を強化
    • Indent-Rainbow - インデントを淡い虹色の背景で表示し、読みやすくする
    • Markdown PDF - MarkdownファイルをPDF、HTML、PNG、JPEGファイルに変換。
    • Sublime Text Keymap - VS Code用のSublime Textキーバインド。
    • SVG Viewer - Visual Studio用SVGビューアー
    • TODO Highlight - TODO、FIXME、その他のアノテーションをハイライト
    • Trailing Spaces - 末尾のスペースをハイライトし、素早く削除
    • TSLint - TSLintをVSCodeに統合。
    • Typora - VSCodeでTyporaのようなMarkdown編集体験を提供。
    • View In Browser - HTMLファイルをシステムのデフォルトブラウザで表示
    • vscode-icons - Visual Studio用アイコン

  • HTML

    • Auto Close Tag - 自動閉じタグ
    • Auto Rename Tag - 対になるHTML/XMLタグを自動的にリネーム
    • HTML Snippets - HTMLスニペット
    • IntelliSense for CSS class names in HTML - ワークスペース内のCSSファイルに基づいて、HTMLのclass属性にCSSクラス名補完を提供
    • HTML CSS Support - HTMLドキュメント用CSSサポート

  • JS

    • Auto Import - 利用可能なすべてのインポートを自動的に検索、解析し、コードアクションを提供。
    • Code Runner - 複数の言語のコードスニペットまたはコードファイルを実行。
    • Import Cost - エディタにimport/requireパッケージのサイズを表示。
    • npm Intellisense - importステートメントでnpmモジュールをオートコンプリート。
    • Path Intellisense - ファイル名をオートコンプリート。
    • Regex Previewer - 正規表現のテストとプレビュー。

  • React/Vue

    • react-beautify - prettydiff/esformatterをラップして、JavaScript、JSX、TypeScript、TSXファイルをフォーマット。
    • Reactjs code snippets - ES6シンタックスでのReactjs開発用コードスニペット。
    • Vetur - VS Code用Vueツール、シンタックスハイライト、オートコンプリート、スニペットなどに対応。

Vim プラグイン
