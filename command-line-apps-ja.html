Awesome Command Line Apps

便利なコマンドラインアプリの厳選リスト

✦ 私のmacOSアプリケーション:

データベース

  • mycli - 自動補完とシンタックスハイライト機能を備えたMySQL用CLI。
  • pgcli - 自動補完とシンタックスハイライト機能を備えたPostgres用コマンドラインインターフェース。

メディア

  • cmus - Unix系オペレーティングシステム向けの小さく、高速で強力なコンソール音楽プレイヤー。
  • youtube-dl - YouTube.comおよびその他のサイトから動画をダウンロード。

開発者向け

  • httpie - モダンなコマンドラインHTTPクライアント。
  • JSON Schema CLI - JSON Schemaを扱うためのCLI。フォーマット、リンティング、テスト、バンドリングなど、ローカル開発からCI/CDパイプラインまでカバー。
  • mdctl - Markdownファイルを処理するためのコマンドラインツール。画像のダウンロード、ファイルの翻訳など。
  • SimTool - 強力なアプリブラウジングとファイル表示機能を備えたiOSシミュレーター管理用ターミナルUI。

その他

  • ack - プログラマー向けに最適化されたgrep風ツール。
  • Airlift - 添付ファイル付きデータをAirtableにアップロード＆マージ。
  • archey-osx - OS X向けのarcheyスクリプト。
  • asciinema - ターミナルセッションを簡単に録画し、ターミナルやWebブラウザで再生。
  • autojump - インテリジェントなautojumpでcdを置き換え。
  • bash-it - bash版のoh-my-zsh。
  • bat - シンタックスハイライトとGit統合を備えたcat(1)クローン。
  • bclm - バッテリーの最大充電量を制限するmacOSコマンドラインユーティリティ。
  • ccat - catと同様に動作するが、シンタックスハイライト付きでコンテンツを表示するカラーリングcat。
  • ClamAV - クロスプラットフォームのオープンソースアンチウイルスエンジン。
  • cmatrix - 映画「マトリックス」にインスパイアされたターミナルスクリーンセーバー。
  • cool-retro-term - 古いブラウン管ディスプレイを模倣した見た目の良いターミナルエミュレーター。
  • CSV2Notion Neo - CSVまたはJSONデータを画像付きでNotionデータベースにアップロード＆マージ。
  • dark-mode - コマンドラインからダークモードを制御。
  • eureka - ターミナルを離れることなくアイデアを入力・保存するCLIツール。
  • Fish - zshに似たスマートでユーザーフレンドリーなシェル。
  • fselect - SQLライクなクエリでファイルを検索。
  • GitHub CLI - ghはコマンドライン上のGitHub。プルリクエスト、イシュー、その他のGitHubの概念を、gitやコードの作業をしているターミナルに持ち込む。
  • Glances - クロスプラットフォームのcursesベースシステム監視ツール。
  • Go2Shell - Finderの現在のディレクトリでターミナルウィンドウを開く。
  • hns - マイクからの音声を直接クリップボードに文字起こしする音声テキスト変換ツール。
  • htop - Unixシステム向けのインタラクティブなテキストモードプロセスビューアー。より良い'top'を目指す。
  • job - 短期的なコマンドを長期的なジョブに変換。
  • lnav - ログファイルナビゲーター。
  • m-cli - macOS用のスイスアーミーナイフ。
  • mac-cleanup-go - プレビューファーストのワークフローを持つmacOSディスククリーンアップTUI：キャッシュと開発アーティファクトをスキャンし、結果をプレビューし、選択したアイテムをゴミ箱に移動。
  • Mac-CLI - Macを管理する究極のツール。OS Xシステムの使用を自動化。
  • Magic Wormhole - コンピューター間で安全にファイルを転送。
  • MarkersExtractor - Final Cut Pro用のマーカーメタデータ抽出・変換ツールおよびライブラリ。
  • mas - Mac App Store用のシンプルなコマンドラインインターフェース。
  • Miller - CSV、TSV、表形式JSONなどの名前付きインデックスデータに対するawk、sed、cut、join、sortのようなツール。
  • ndm - ソファから直接npmを管理。
  • nushell - Rustで書かれたモダンなGitHub時代のシェル。
  • nvm - 複数のアクティブなnode.jsバージョンを管理するPOSIX準拠のbashスクリプト。
  • OpenBoot - Homebrewパッケージ、Cask、dotfiles、シェル設定、git設定、macOS環境設定をキャプチャ・復元するMac開発環境マネージャー。
  • OpenRecall - プライバシーを維持しながら、デジタル履歴にアクセスし、記憶と生産性を向上。
  • Rebound - コンパイルエラーが発生した際にターミナルで即座にStack Overflowの結果を閲覧。
  • renamed.to - ドキュメントの内容とOCRを使用したAI搭載のバルクファイルリネームCLI。Homebrewでインストール可能（brew tap renamed-to/cli）。
  • ripgrep (rg) - ack、ag、grepに似た（しかしより高速な）非常に高速なテキスト検索ツール。
  • rustnet - プロセス識別とディープパケットインスペクションを備えたクロスプラットフォームのネットワーク監視TUI。
  • Serial - Mac用のフル機能シリアルターミナル。
  • shallow-backup - インストール済みアプリケーション、dotfilesなどのテキストドキュメントを簡単に作成。
  • ShellHistory - シェル履歴のバックアップ、同期、整理。
  • silver searcher (ag) - 速度を重視したackに似たコード検索ツール。
  • spaceship - 宇宙飛行士のためのZshプロンプト。
  • Taskbook - コマンドライン環境のためのタスク、ボード、メモ。
  • Television - 非常に高速な汎用ファジーファインダーTUI。
  • TextQL - CSVやTSVなどの構造化テキストに対してSQLを実行。
  • thefuck - 前のコンソールコマンドのエラーを修正する面白いソフトウェア。
  • tldr - コンソールコマンドの共同チートシート。
  • tmux - 1つのターミナルから複数のターミナルを管理するターミナルマルチプレクサー。
  • tmuxinator - 複雑なtmuxセッションを簡単に管理。
  • trash - ファイルやディレクトリをゴミ箱に移動。
  • VisiData - データの発見と整理のためのターミナルスプレッドシートマルチツール。
  • xonsh - Pythonベースのクロスプラットフォーム、Unix志向のシェル言語＆コマンドプロンプト。
  • Zsh - 対話的な使用のために設計されたシェル。強力なスクリプト言語でもある。
  • ohmyzsh - zsh設定を管理するための素晴らしいコミュニティ主導フレームワーク。
  • xcodes - 複数バージョンのXcodeをインストール・切り替えするコマンドラインツール。
  • xxh - SSH経由でどこにでもお気に入りのシェルを持ち運び。
  • x-cmd - Unixの哲学に基づいた膨大で興味深いツールコレクション。
  • zellij - バッテリー同梱のターミナルワークスペース（tmuxなどのモダンな代替）。
